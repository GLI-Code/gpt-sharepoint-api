Política de Privacidad – API SharePoint (GPT Suerox)

Esta API fue desarrollada con el propósito de recuperar archivos específicos alojados en SharePoint a solicitud del usuario mediante una interfaz GPT personalizada.

1. Datos que recopilamos

Esta API no almacena información personal identificable. Solo procesa los siguientes datos de forma temporal:

  • Nombre del archivo solicitado
  • Ruta de la carpeta en SharePoint

No se almacenan consultas ni respuestas.

2. Uso de los datos

Los datos se usan exclusivamente para consultar archivos en una cuenta autorizada de SharePoint mediante Microsoft Graph API.

3. Acceso a los datos

Solo usuarios autenticados mediante un token de seguridad (Bearer Token) pueden acceder a la API.

4. Seguridad

  • Se usa HTTPS para cifrado en tránsito
  • Se requiere autenticación OAuth2 para acceder a SharePoint
  • Se valida un token de API para acceder al endpoint

5. Conservación de datos

Esta API no almacena datos de forma persistente.

6. Contacto

Si tienes dudas sobre esta política, contáctanos en: privacidad@tudominio.com
